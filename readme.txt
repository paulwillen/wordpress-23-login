== Description ==

This plugin changes the login form from you WordPress install to the WordPress 2.3 version. 

This, because the login form of WordPress 2.3 is the best-looking login box in WordPress History

== Installation ==

You can install this plugin via the WordPress plugin admin section. Or you can upload the plugin to you ftp in wp-content/plugins.

After activation the login form will display the WordPress 2.3 version login form.
